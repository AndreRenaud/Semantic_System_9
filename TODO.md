# System 9 CSS Tasks

A list of discrete tasks to create a classless semantic CSS file that styles HTML to look like Apple Macintosh System 9.

- [ ] **Typography & Base Styles**
  - [x] Set default font to Chicago.
  - [ ] Set base font size, line height, and text color (black).
  - [ ] Set page background (classic System 9 gray or desktop pattern).

- [ ] **Headings & Text Elements**
  - [ ] Style `h1` through `h6` to match System 9 dialog/window text.
  - [ ] Style links (`a`) with classic blue, underline, and active states.
  - [ ] Style `hr` to look like a classic etched separator line.
  - [ ] Style `blockquote`, `code`, `kbd`, and `samp`.

- [ ] **Buttons**
  - [ ] Style `<button>`, `<input type="submit">`, `<input type="button">`, and `<input type="reset">`.
  - [ ] Add rounded corners (pill shape for default buttons).
  - [ ] Add classic double-border for default/primary buttons.
  - [ ] Add active (pressed) states with inverted colors or dark backgrounds.

- [ ] **Form Inputs**
  - [ ] Style text inputs (`<input type="text">`, `<input type="password">`, `<input type="email">`, etc.) with inset borders.
  - [ ] Style `<textarea>` with inset borders and Chicago font.
  - [ ] Style `<select>` dropdowns to mimic System 9 menus.
  - [ ] Style `<input type="checkbox">` with the classic square and cross/check.
  - [ ] Style `<input type="radio">` with the classic circle and dot.
  - [ ] Style `<fieldset>` and `<legend>` to look like System 9 group boxes.

- [ ] **Tables**
  - [ ] Style `<table>`, `<th>`, and `<td>` with classic solid borders.
  - [ ] Add alternating row colors or simple header backgrounds if appropriate.

- [ ] **Lists**
  - [ ] Style `<ul>`, `<ol>`, and `<dl>` with appropriate spacing.
  - [ ] Customize list markers if necessary.

- [ ] **Interactive & Semantic Elements**
  - [ ] Style `<progress>` and `<meter>` to look like System 9 loading bars.
  - [ ] Style `<details>` and `<summary>` to look like classic disclosure triangles.
  - [ ] Style `<dialog>` to look like a System 9 alert or modal window (with title bar and borders).

- [ ] **Containers (Optional/Semantic)**
  - [ ] Style `<section>` to look like System 9 windows with title bars and close boxes.
  - [ ] Style scrollbars (using `::-webkit-scrollbar`) to mimic System 9 scrollbars with arrows and proportional thumbs.
